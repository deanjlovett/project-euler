# project-euler-0016-power-digit-sum

Project Euler

Power digit sum

Problem 16

https://projecteuler.net/problem=16

2^15 = 32768 and the sum of its digits is 3 + 2 + 7 + 6 + 8 = 26.

What is the sum of the digits of the number 2¹⁰⁰⁰ ( 2^1000 )?
